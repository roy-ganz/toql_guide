# Guide for Toql 

This is a user guide for  [Toql](https://docs.rs/toql/0.3/toql/), an ORM for async databases.

## Build

This guide uses mdbook.

To build the `/src` folder and put the guide into `/docs` run:

```bash
mdbook build
```
